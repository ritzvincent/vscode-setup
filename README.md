# VS Code Setup

My current setup for Visual Studio Code for Web Development.

## Extensions

1. [`Dracula`](https://github.com/dracula/visual-studio-code) - Theme
2. [`ESLint`](https://github.com/eslint/eslint) - For Linting and style checking in JavaScript
3. [`markdownlint`](https://github.com/DavidAnson/markdownlint) - For linting and style checking when writing READMEs and documentations using markdown.
4. [`Prettier`](https://github.com/prettier/prettier-vscode) - To automatically format, `HTML5`, `CSS3`, `JSON`, `JavaScript`, `MD` and other files used in web development.
5. [`Live Server`](https://github.com/ritwickdey/vscode-live-server) - To host HTML files locally.
6. [`Simple Icons`](https://github.com/sjml/vscode-simple-icons) - To spice up file and folder icons.
